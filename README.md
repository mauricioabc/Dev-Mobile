# Dev-Mobile
Repositório de React Native.

# Como rodar?

Cria o projeto: 

```bash
npx create-expo-app NomeDoProjeto
```
Instala o yard:

```bash
npm install yarn add expo
```
Roda o servidor com o projeto:

```bash
npx expo start
```
